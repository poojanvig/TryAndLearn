C Strings

Strings

In C programming, the one-dimensional array of characters are called strings, which is terminated by a null character '\0'.

String Decleration

There are two ways to declare a string in C programming:

1. Through an array of characters.

Example: 
                
            char name[6];

2. Through pointers.

Example: 
                
                
            char *name;

String Initialization

There are two ways to initialize a string in C programming:

1. Arrays

Example:
                
            char name[6] = {'C', 'l', 'o', 'u', 'd', '\0'};

2. Pointers

Example:
                
            char name[] = "Cloud";

Repersentation Of String

                
            #include

                int main ()
                {
                   char name[6] = {'C', 'l', 'o', 'u', 'd', '\0'};
                
                   printf("Tutorials%s\n", name );
                
                   return 0;
                }
Program Output:
TutorialsCloud

C String Functions

There are many important string functions defined in "string.h" library.
  • strlen(string_name)
    • returns the length of string name.
  • strcpy(destination, source)
    • copies the contents of source string to destination string.
  • strcat(first_string, second_string)
    • concats or joins first string with second string. The result of the string is stored in first string.
  • strcmp(first_string, second_string)
    • compares the first string with second string. If both strings are same, it returns 0.
  • strrev(string)
    • returns reverse string.
  • strlwr(string)
    • returns string characters in lowercase.
  • strupr(string)
    • returns string characters in uppercase.
